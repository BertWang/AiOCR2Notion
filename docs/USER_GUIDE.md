## 基礎功能

### 上傳筆記

1. 點擊上傳區域或拖曳檔案
2. 選擇圖片 (JPG, PNG, WEBP)
3. 點擊「開始智慧歸檔」
4. 等待 AI 處理完成

**提示**:
- 圖像清晰度越高，OCR 效果越好
- 支持手寫稿、掃描檔和截圖
- 單次上傳最大 10MB

### 編輯筆記

在分割編輯器中：

1. **左側**: 原始圖像 + 放大縮小
2. **右側**: 編輯 Markdown 或預覽
3. **標籤**: 添加/移除分類標籤
4. **評分**: 為筆記質量評分

**快捷鍵**:
- `Ctrl+S` / `Cmd+S`: 保存
- `Tab`: 切換編輯/預覽

### 批次管理

選中多個筆記後：
- ✨ **合併**: 將多個筆記整合為一
- 📥 **匯出**: 下載為 Markdown
- 🔄 **同步 Notion**: 發送到 Notion
- 🗑️ **刪除**: 批次刪除（謹慎！）

## MCP 市場

### 安裝服務

1. **瀏覽市場** 標籤
2. 搜尋或篩選服務
3. 點擊「安裝」按鈕
4. 填寫必需配置（API 密鑰等）
5. 確認安裝

### 管理已安裝服務

在 **已安裝** 標籤下：

- **測試連線**: 驗證服務連接
- **啟用/停用**: 控制服務狀態
- **顯示詳情**: 查看配置和診斷信息

### 評分和收藏

- ⭐ **評分**: 點擊星星評分 (1-5)
- ❤️ **收藏**: 點擊心形按鈕

### 服務配置驗證

展開服務詳情時，查看：
- ✓ 配置完整性
- ✓ 認證信息有效性
- ✓ 端點可訪問性
- ✓ 連接健康狀態

### 故障排查

如果服務連接失敗：

1. 點擊「顯示詳情」
2. 查看「故障排查」面板
3. 檢查建議的修復步驟
4. 更新配置後重新測試

### 智能故障轉移

系統自動檢測故障：
- 自動切換到備用服務
- 記錄轉移事件
- 支持手動觸發轉移

## 搜尋功能

### 全文搜尋

1. 在搜尋框輸入關鍵詞
2. 查看建議下拉列表
3. 選擇結果或按 Enter

**支持的搜尋類型**:
- 筆記標題和內容
- 標籤
- 日期範圍
- 快速搜尋集合

### 高級搜尋

```
tag:會議 2026-01  # 2026年1月的會議筆記
status:completed  # 已完成的筆記
title:計畫       # 標題包含「計畫」
```

### 保存搜尋

點擊 ⭐ 保存常用搜尋，下次快速調用。

## 高級功能

### 筆記相關性分析

系統自動找出相關筆記：
- 基於內容相似度
- 基於標籤匹配
- 基於時間接近性

### 知識圖譜

可視化你的筆記關係網絡：
- **節點**: 筆記
- **邊**: 相關性連接
- **顏色**: 按分類編碼

### AI 建議

系統為每個筆記生成：
- 改進建議
- 標籤建議
- 相關資源

點擊 ✨ 查看詳細建議。

## 常見問題

### Q: 上傳後多久能完成 OCR？

A: 通常 2-5 秒。如果超過 30 秒，檢查網絡連接或重新上傳。

### Q: 如何提高 OCR 準確率？

A: 
- 確保圖像清晰（300 DPI 以上）
- 光線充足，無陰影
- 手寫盡量整潔
- 選擇高級 OCR 提供商

### Q: 可以導出筆記嗎？

A: 支持多種格式：
- Markdown (.md)
- PDF
- Word (.docx)
- Notion 導出

### Q: 支持多用戶嗎？

A: 目前是單用戶 MVP。多用戶支持計劃在 v2.0。

### Q: 數據安全嗎？

A: 
- 本地存儲（默認）
- 圖像存儲在 `/public/uploads`
- 支持端到端加密（企業版）

### Q: 如何導入舊筆記？

A: 
1. 準備圖片文件
2. 使用批次上傳功能
3. 系統自動 OCR 處理

### Q: 成本是多少？

A: 
- 開源免費版本
- 企業版按使用量計費
- 詳見定價頁面

### Q: 如何備份？

A:
```bash
# 導出所有筆記
npm run export

# 備份數據庫
npx prisma db dump > backup.sql
```
